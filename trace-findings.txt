=== CRITICAL ISSUE #1: app/api/analyze/route.ts ===
Line 3: `const client = new Anthropic();`
This creates an Anthropic client at MODULE LOAD TIME.
If ANTHROPIC_API_KEY is not set, this will throw immediately when the module is imported.
Even if the key IS set, this route creates its own client separate from analyze.ts.
This route is NOT used by the main scan pipeline (scan/trigger uses lib/analyze.ts directly).
But if anything imports or hits /api/analyze, it will crash without the key.
STATUS: MEDIUM RISK — only affects direct /api/analyze calls, not the main scan flow.
FIX: Lazy-initialize the client.
